# Tritubot Web

Landing page construida con Next.js para presentar una versión adaptada y mejorada de la propuesta digital de Tritubot, lista para desplegar en Vercel.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4 (vía PostCSS)

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run lint`

## Despliegue en Vercel

1. Importa este repositorio en Vercel.
2. Mantén la configuración por defecto para Next.js.
3. Ejecuta el deploy; no se requiere configuración adicional.

## Notas

- El contenido actual reinterpreta la propuesta del sitio de referencia y deja la base preparada para reemplazar textos, métricas, logos o formularios reales.
- El scaffold se hizo manualmente en la carpeta actual porque `create-next-app` falla cuando el nombre de la carpeta contiene espacios y mayúsculas.
